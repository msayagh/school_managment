# Shared Utilities

This directory contains shared utility functions and helpers used across multiple microservices.

## Purpose
- Reusable code across services
- Common validation functions
- Shared error handling
- Date/time utilities
- Logging utilities

## Utilities to be Implemented

### Validation Utilities
- Email validation
- Phone number validation
- Date range validation
- Capacity validation

### Error Handling
- Standard error response format
- Error codes and messages
- Error logging

### Date/Time Utilities
- Date formatting
- Timezone conversion
- Schedule conflict detection

### Logging Utilities
- Structured logging
- Log levels (debug, info, warn, error)
- Request/response logging

### Authentication Utilities
- JWT token generation and validation
- Password hashing
- Role-based access control helpers

## Usage
Services can import these utilities to maintain consistency and reduce code duplication.
