# Bookings Service

## Overview
Microservice responsible for managing room bookings and reservations.

## Responsibilities
- Room booking creation and management
- Booking conflict resolution
- Booking cancellation
- Booking history and reporting

## API Endpoints (To Be Implemented)
- `POST /api/bookings` - Create a new booking
- `GET /api/bookings` - Get all bookings
- `GET /api/bookings/:id` - Get booking by ID
- `PUT /api/bookings/:id` - Update booking information
- `DELETE /api/bookings/:id` - Cancel booking
- `GET /api/bookings/room/:roomId` - Get bookings for a specific room
- `GET /api/bookings/conflicts` - Check for booking conflicts
