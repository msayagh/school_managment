# Rooms Service

## Overview
Microservice responsible for managing school rooms and facilities.

## Responsibilities
- Room/facility creation, editing, and deletion
- Room capacity management
- Room availability tracking
- Room information retrieval

## API Endpoints (To Be Implemented)
- `POST /api/rooms` - Create a new room
- `GET /api/rooms` - Get all rooms
- `GET /api/rooms/:id` - Get room by ID
- `PUT /api/rooms/:id` - Update room information
- `DELETE /api/rooms/:id` - Remove room
- `GET /api/rooms/:id/availability` - Check room availability
- `GET /api/rooms/available` - Get available rooms for a time slot
