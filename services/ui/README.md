# UI Service

## Overview
Frontend application for the school management system, providing user interfaces for admins, teachers, and students.

## Responsibilities
- Admin dashboard for school setup and management
- Student registration and profile management interface
- Teacher management interface
- Activity creation and management interface
- Room and booking management interface
- User authentication and authorization UI
